This README defines the usage of all the D3 html,js and css files

- bar_chart_bfa - represents the BFA. takes an input JSON file for the same
- scatter_plot - represents the BFC analysis. takes an input JSON file for the same
- pie_chart - used to show the domain diversity of file types in Tika.
- cross_correlation - represents the data as a matrix. takes an input CSV file for the same
- fht_analysis - represents the file header trailer as a matrix. takes an input CSV file for the same

all the JSONs and CSVs can be found in "JSONs,CSVs for BFA,BFC,BFCC,FHT" folder
